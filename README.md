## Create LESS file

RockSkinUikit will look for the less file in `/site/assets/RockSkinUikit/theme.less`.

## Using LESS variables

You can modify LESS variables from PHP by setting the `$config->lessVars` array:

```php
$config->lessVars = [
  'tm-primary' => '#146dbf',
];
```
